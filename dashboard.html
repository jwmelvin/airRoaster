Commissioning sequence
  1. Flash. Not a WebSocket command — run on the host with the board connected: ./verify.sh upload
  2. Sanity. Stay in manual; confirm sensors/OLED/Artisan behave. After a few minutes read the load counters (jitter and worst loop pass, µs).
  3. Characterize. Set fan, set a moderate heat, let it settle, then step-test. Fan Heat Read the result; eyeball dT/Kp/tau/theta for sanity, and check it settled.
  4. Apply gains. Tight suggestion, or the conservative set if tight looks twitchy.
  5. Calibrate feedforward. Still steady: set ambient (blank = cold junction), then calibrate. Amb °C
  6. Close the loop. Set the inlet setpoint near the current inlet temp (bumpless), then nudge it and watch tracking on the chart above. °C
  7. The real test. With the loop holding, deliberately change fan speed and confirm inlet temp barely moves — feedforward earning its keep. Tune from there. Fan
